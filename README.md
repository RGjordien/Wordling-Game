# Wordle-Game
Game berbasis wordle, namun ditambahkan dengan minigame dimana setiap huruf yang ditemukan sesuai/hijau akan memunculkan soal matematika +,-,×,÷ . Jika dijawab benar, akan diberikan hint huruf di huruf ke-random, jika salah, point akan dikurang. Dikasih 6 kesempatan buat nebak kata berjumlah harus 5 huruf.
